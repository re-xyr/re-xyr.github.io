Call matrix is a technique used in the minimal foetus language to determine whether a program terminates in a predictable way. In short, a recursive call, i.e. a call from a function to itself would terminate if (but not only if) the call reduces at least one of the arguments, that is, the arguments is eliminated by pattern matching, projecting or applying. One program terminates, if all calls in it terminates. (Nov 27, 2020)

Martin-Löf type theory is the dependently typed lambda calculus equipped with the infamous Id\rm IdId type, which is the type of identity, or equality, given that it is an equivalence and satisfies the Leibniz property. This type is used as a basis of the path in the original homotopy type theory, but the focus has moved to one constructive interpretation of the HoTT, namely the cubical type theory, where a path is indeed a function p:[0,1]→Ap : [0, 1] \to Ap:[0,1]→A.
